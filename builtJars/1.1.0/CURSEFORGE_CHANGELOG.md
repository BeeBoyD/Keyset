# Keyset 1.1.0

`1.1.0` is the new active Keyset line for Minecraft `1.20.1+`.

## Highlights

- Better modern UI scaling on small windows and higher GUI scales
- Fixed overlapping card text and compact conflict-row clipping in the `1.20.1+` UI
- Better Controls-screen integration with no duplicate Keyset buttons on reopen or GUI scale changes
- Safer config recovery with backup restore and broken-file archiving
- Manual `Controls` edits now sync back into the active profile with visible status

## Support policy

- `1.1.x` targets Minecraft `1.20.1+`
- Older `1.16.5-1.19.4` builds stay on the `1.0.x` line for critical bug fixes only
- Profile compatibility stays the same across the supported release line
