# Keyset 1.0.0

Keyset is now on a stable `1.0.0` release line.

This release focuses on making the mod much easier to understand day to day: all supported loaders benefit from the clearer wording around live edits and Safe Fix, while the modern `1.20.1+` UI line now has a cleaner dashboard layout and more stable responsive behavior across window sizes.

## Highlights

- New dashboard-style Keyset screen on the modern `1.20.1+` releases
- Cleaner labels, shorter actions, and clearer workflow text
- Responsive layout fixes for narrow and short windows
- Better conflict navigation and safer preview/apply flow for cleanup
- Refreshed public docs and release packaging for the stable launch

## Compatibility

- Fabric / Quilt: `1.16.5` through `1.21.11`
- Forge: `1.16.5` through `1.21.1`
- NeoForge: `1.20.1` through `1.21.11`

## Notes

- Legacy `1.16.5-1.19.4` builds keep the older screen layout, but stay on the same feature set and profile format.
- Quilt uses the Fabric jars.
